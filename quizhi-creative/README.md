## Function Nodes (Design)

### 1. Builder Node
- **Input**: User goal, required tools, constraints.
- **Process**: LLM-based generation of skill logic.
- **Output**: JSON/Markdown definition of the skill.

### 2. Export Node
- **Input**: Skill definition.
- **Process**: Render to graphical format (e.g., Mermaid diagram, PNG).
- **Output**: Downloadable file.
