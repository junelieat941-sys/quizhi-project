# Quizhi Project

This project aims to replicate the functionality of the "Agent Skills Guide" app, providing a comprehensive tool for learning and creating Agent Skills.

## Structure

- **quizhi-creative**: The core application, featuring a skill generator, concept guide, and graphical export capabilities.

## Features

1.  **Skill Generator**: Step-by-step wizard to create custom Agent Skills.
2.  **Concept Guide**: Interactive tutorials on Agent Skills principles.
3.  **Graphical Export**: Export your skill designs as images or structured data.
